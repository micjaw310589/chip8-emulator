# chip8-emulator

Chip-8 emulator/interpreter. Educational purposes.

# Pre-requirements

SDL3 library is required to build and compile this project. See [SDL3 github](https://github.com/libsdl-org/SDL.git) for more info.

# How to run

./chip8_emulator \[path_to_rom\] \[clock_hz\] \[window_heigh_px\]
